An application that provides investment insights by summarizing recent market trends, explaining financial terms. User queries are answered based on real-time data using AI agent service

---

- **Repo link:-** https://github.com/yashksaini-coder/investo-glow

- **Live link:-** https://investo-glow.vercel.app/

### Server

Backend API server using fastapi in python to create and host custom API endpoint for the investo-glow Next.js application. The server is created using the following Tech stack:

- Fastapi (Python service scripts)
- Docker (deployment)
- Agno (AI Agent)
- Render (deployment)
- Redis (caching)
