**C** is general purpose programming language with features of expression, modern flow control and Data Structures as well.

- **Creator**:- *Dennis-Ritchie*
- Originally designed for and implemented on the UNIX OS. The entire OS, C compiler and all the essentially materials and concepts of UNIX applications are written in C. 
- It is not tied to any particular hardware or system, however it is easy to write programs that will run without change on any machine that supports C.
- It is a pleasant versatile (having many different uses) language for a wide variety of programs.

### Perks:-

- Versatile
- Easy to Learn
- Helps build low level logic.
